# Lexical Analyzer (Mini-lex)

## Overview
This is a simple Lexical Analyzer built in C++. It parses input text and breaks it down into meaningful tokens like Keywords, Operators, and Identifiers, which is the first phase of a compiler.

## Features
- Tokenization of basic C++ like syntax.
- Classification of tokens.
- Error handling for unknown characters.

## How to Run
- Compile the .cpp file.
- Provide a string or file as input.
- View the list of generated tokens in the console.
